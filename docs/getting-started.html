Getting Started with Pseudocode

Learn Pseudocode in 10 minutes ‚Äî from zero to productive

Pseudocode is a high-performance programming language designed to look like the pseudocode you'd write in a textbook. It's fast (2-5x faster than Python), intuitive, and comes with 80+ built-in functions for common tasks.

üí° Why Pseudocode?

If you've ever written pseudocode in a algorithms class and wished it would just run ‚Äî that's exactly what this language does. No semicolons, no braces, just clear readable code.

Installation

There are two ways to get started with Pseudocode:

Option 1: VS Code Extension (Recommended)

The easiest way to get started. Install the VS Code extension for full IntelliSense, debugging, and one-click execution.

  1. Install VS Code

    Download from code.visualstudio.com

  2. Install the Extension

    Search for "Pseudocode" in the Extensions panel, or install directly.

  3. Create a .pseudo file

    Create a new file with the .pseudo extension and start coding!

Option 2: Build from Source

For advanced users who want the command-line compiler:

Terminal
git clone https://github.com/NagusameCS/Pseudocode.git
cd Pseudocode/cvm
make
./pseudo examples/hello.pseudo

Hello World

Let's write your first Pseudocode program:

hello.pseudo
// Your first Pseudocode program
print("Hello, World!")

Run it with Ctrl+Shift+R in VS Code, or from the terminal:

Terminal
./pseudo hello.pseudo

Output:

Output
Hello, World!

Variables

Use let to declare variables. Pseudocode is dynamically typed.

variables.pseudo
// Declare variables with 'let'
let name = "Alice"
let age = 25
let pi = 3.14159
let is_student = true

print("Name: " + name)
print("Age: " + str(age))

// Reassign without 'let'
age = 26
print("Happy birthday! Now: " + str(age))

Functions

Define functions with fn and close with end:

functions.pseudo
// Define a function
fn greet(name)
    print("Hello, " + name + "!")
end

// Call it
greet("World")

// Function with return value
fn add(a, b)
    return a + b
end

let result = add(5, 3)
print(result)  // 8

// Recursive function
fn factorial(n)
    if n <= 1 then
        return 1
    end
    return n * factorial(n - 1)
end

print(factorial(5))  // 120

Conditionals

Use if/then/end, elif, and else:

conditionals.pseudo
let score = 85

if score >= 90 then
    print("A - Excellent!")
elif score >= 80 then
    print("B - Good job!")
elif score >= 70 then
    print("C - Passing")
else
    print("Need improvement")
end

// Logical operators
let age = 20
let has_id = true

if age >= 18 and has_id then
    print("Access granted")
end

Loops

For Loops

Iterate over ranges or arrays with for/in/do:

loops.pseudo
// Range loop (0 to 4)
for i in 0..5 do
    print(i)
end

// Loop over array
let fruits = ["apple", "banana", "cherry"]
for fruit in fruits do
    print("I like " + fruit)
end

// Range with step (using range function)
for i in range(0, 10, 2) do
    print(i)  // 0, 2, 4, 6, 8
end

While Loops

while.pseudo
let count = 0

while count < 5 do
    print("Count: " + str(count))
    count = count + 1
end
‚ö° Performance Tip

For loops with numeric ranges are automatically JIT-compiled to native machine code. This makes them as fast as C for compute-intensive operations!

Pattern Matching

Use match/case for elegant branching:

match.pseudo
fn describe_number(n)
    match n
        case 0 then
            return "zero"
        case 1 then
            return "one"
        case n if n < 0 then
            return "negative"
        case _ then
            return "positive"
    end
end

print(describe_number(0))   // zero
print(describe_number(-5))  // negative
print(describe_number(42))  // positive
