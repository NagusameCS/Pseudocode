Install Pseudocode

Get up and running in minutes with the VS Code extension or build from source

VS Code Extension

Recommended

The easiest way to get started. The VS Code extension includes syntax highlighting, IntelliSense, debugging support, and a built-in REPL.

  1. Open VS Code

    Launch Visual Studio Code on your computer

  2. Open Extensions View

    Press Ctrl+Shift+X (Windows/Linux) or Cmd+Shift+X (macOS)

  3. Search for Pseudocode

    Type "Pseudocode" in the search box

  4. Install

    Click the Install button on the "Pseudocode Language" extension

Open in VS Code
Note: The extension will prompt you to install the Pseudocode runtime the first time you run a .pseudo file. Follow the instructions in VS Code.

Build from Source

Build the Pseudocode interpreter from source for the best performance and latest features. This method gives you access to the JIT compiler and all optimization flags.

Requirements

  • GCC or Clang (C11 compatible)
  • Make
  • LuaJIT (for DynASM, bundled)
  • Git

  1. Install dependencies

    sudo apt update
sudo apt install build-essential git luajit

  2. Clone the repository

    git clone https://github.com/NagusameCS/Pseudocode.git
cd Pseudocode/cvm

  3. Build the interpreter

    make

  4. Install globally (optional)

    sudo cp pseudo /usr/local/bin/

  1. Install Homebrew (if not installed)

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Install dependencies

    brew install luajit

  3. Clone and build

    git clone https://github.com/NagusameCS/Pseudocode.git
cd Pseudocode/cvm
make

  4. Install globally (optional)

    sudo cp pseudo /usr/local/bin/

  1. Install WSL

    Open PowerShell as Administrator and run:

    wsl --install

  2. Open Ubuntu in WSL

    Install dependencies and build as shown in the Linux tab above

Docker

Run Pseudocode in a containerized environment without installing anything locally.

  1. Pull and run

    docker run -it --rm -v $(pwd):/code nagusame/pseudocode

  2. Run a script

    docker run --rm -v $(pwd):/code nagusame/pseudocode /code/script.pseudo

Verify Installation

After installation, verify everything is working correctly:

  1. Check version

    pseudo --version

    You should see the Pseudocode version number

  2. Create a test file

    Create a file called hello.pseudo with:

    print("Hello, Pseudocode!")

  3. Run the file

    pseudo hello.pseudo

    You should see: Hello, Pseudocode!

  4. Try JIT compilation (optional)

    pseudo -j hello.pseudo

    The -j flag enables the JIT compiler for faster execution
