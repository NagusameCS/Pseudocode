Language Reference

Complete documentation for the Pseudocode programming language.

Introduction

Pseudocode is a simple, fast programming language designed to be as readable as possible. It compiles to bytecode and runs on a high-performance virtual machine written in C.

Key features:

  • Intuitive syntax â€” reads like natural language
  • Blazing fast â€” faster than native Python
  • Dynamic typing â€” no type annotations needed
  • First-class functions â€” supports recursion and closures
  • Built-in arrays â€” with push, pop, and slicing

Installation

Build the C VM from source:

# Clone the repository
git clone https://github.com/yourusername/Pseudocode.git
cd Pseudocode/cvm

# Build (requires GCC or Clang)
make

# Run a program
./pseudo ../examples/hello.pseudo

Hello World

Create a file called hello.pseudo:

print("Hello, World!")

Run it:

./pseudo hello.pseudo

Variables

Declare variables using the let keyword:

let name = "Alice"
let age = 25
let pi = 3.14159
let active = true

Variables are dynamically typed and can be reassigned:

let x = 10
x = "now a string"  // Valid!

If Statements

Conditional execution with if, elif, and else:

if score >= 90 then
    print("A")
elif score >= 80 then
    print("B")
elif score >= 70 then
    print("C")
else
    print("F")
end

Match Expressions (Pattern Matching)

Pattern matching with match, case, and else:

fn day_name(day)
    match day
        case 1 then
            print("Monday")
        case 2 then
            print("Tuesday")
        case 3 then
            print("Wednesday")
        else
            print("Other day")
    end
end

Match works with strings too:

fn handle_cmd(cmd)
    match cmd
        case "start" then
            print("Starting...")
        case "stop" then
            print("Stopping...")
        else
            print("Unknown command")
    end
end

While Loops

Loop while a condition is true:

let i = 0
while i < 5 do
    print(i)
    i = i + 1
end

For Loops

Iterate over a range of numbers:

// Prints 0 to 9
for i in 0..10 do
    print(i)
end
ðŸ’¡ Tip

Ranges are exclusive of the end value. 0..5 iterates 0, 1, 2, 3, 4.

Defining Functions

Use fn and end to define functions:

fn greet(name)
    print("Hello, " + name + "!")
end

greet("Alice")  // Hello, Alice!

Functions can return values:

fn square(x)
    return x * x
end

print(square(5))  // 25

Recursion

Functions can call themselves:

fn factorial(n)
    if n <= 1 then
        return 1
    end
    return n * factorial(n - 1)
end

print(factorial(10))  // 3628800

Arrays

Create arrays with square brackets:

let numbers = [1, 2, 3, 4, 5]
let mixed = [1, "two", true, nil]
let empty = []

Access elements by index (0-based):

print(numbers[0])   // 1
print(numbers[-1])  // 5 (last element)

numbers[0] = 10     // Modify element

Strings

Strings are immutable sequences of characters:

let greeting = "Hello"
let name = "World"

// Concatenation
print(greeting + ", " + name)  // Hello, World

// Length
print(len(greeting))  // 5

// Indexing
print(greeting[0])    // H
